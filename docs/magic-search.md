# Magic Search 소개

## 배경
정적 페이지로 운영되는 블로그에서도 자연어 질문 기반의 탐색 경험을 제공하기 위해 Magic Search를 추가했습니다. 서버 없이도 동작해야 하므로, 기존에 사용 중인 `/index.json` 검색 인덱스를 활용해 브라우저에서만 동작하도록 설계했습니다.

## 동작 방식 (요약)
1. 사이드바의 **Magic Search** 버튼을 눌러 전용 페이지(`/magic-search/`)로 이동합니다.
2. 질문을 입력하면 브라우저가 `/index.json`을 로드하고, Fuse.js로 관련 글을 검색합니다.
3. 상위 결과를 요약 형태로 구성해 화면에 표시하고, 하단에 참고 링크 목록을 제공합니다.

## 구성 요소
- 검색 인덱스: Hugo가 생성하는 `/index.json`
- UI 템플릿: `layouts/magic-search/single.html`
- 진입 버튼: `layouts/partials/sidebar.html`

## 참고 사항
- 모든 로직이 클라이언트에서 실행되므로 서버 비용이 들지 않습니다.
- 결과 품질은 인덱스에 포함된 요약/본문 길이에 따라 달라집니다.
