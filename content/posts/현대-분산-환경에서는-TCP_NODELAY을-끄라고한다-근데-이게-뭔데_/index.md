# 현대 분산 환경에서는 TCP_NODELAY을 끄라고한다 근데 이게 뭔데?

> **Summary**
> 과거의 네이글 알고리즘과 지연된 확인 응답의 충돌로 인해 현대의 빠른 네트워크 환경에서는 TCP_NODELAY 설정을 활성화해야 한다. 이는 데이터 전송의 지연을 줄이고, 실시간 통신의 성능을 향상시키기 위한 필수적인 조치이다. 현대 시스템에서는 데이터 크기가 크고, 지연 시간이 짧기 때문에 과거의 최적화 기법이 오히려 성능 저하를 초래할 수 있다.
