# AI플랜모드 시각화

> **Summary**
> 브라우저 기반 UI에서 에이전트가 생성한 코드 계획을 표시하고 리뷰할 수 있으며, 팀원과 쉽게 공유하고 피드백을 전송할 수 있다. 다양한 주석 기능과 함께 승인된 계획은 외부 노트 앱에 자동 저장된다. Plannotator UI를 통해 주석 달기, 승인 및 변경 요청 시 구조화된 피드백이 전달된다. 맥, 리눅스, 윈도우에서 설치 가능하다.

---

![Image](image_23a727952ea2.png)

- **Claude Code** 및 **OpenCode**와 통합되어, 에이전트가 생성한 코드 계획을 **브라우저 기반 UI**에서 표시하고 리뷰 가능
- 팀원들에게 쉽게 공유하고, 원클릭으로 피드백 전송도 지원
- 계획 단계에서 **삭제·삽입·교체·코멘트** 등 다양한 주석을 달 수 있음
- 승인 시 에이전트가 구현을 진행하고, 수정 요청 시 **구조화된 피드백 데이터**로 자동 전송
- **/plannotator-review** 명령으로 **git diff 기반 코드 리뷰** 수행 
  - **라인 단위 주석**, **분할/통합 뷰 전환**, **피드백 전송** 기능 지원
- 이미지 피드백 기능 제공 
  - 펜, 화살표, 원형 도구로 시각적 주석 추가 가능
- 승인된 계획은 **Obsidian**, **Bear Notes** 등 외부 노트 앱에 자동 저장
- 맥/리눅스/윈도우(WSL, Powershell) 에 설치 가능
- **동작 방식** 
  - Plannotator UI 를 브라우저에서 열기
  - 계획에 대해서 주석달기(삽입/삭제/대체/댓글)
  - **승인** → 에이전트가 구현 시작
  - **변경 요청** → 입력한 주석이 구조화된 피드백으로 전달
