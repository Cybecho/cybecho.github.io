# 윈도우 탐색기 투명하게

> **Summary**
> Ctrl + Mouse 3을 사용하여 윈도우의 투명도를 조절하고, Win + Mouse 3으로 원래 상태로 복원할 수 있으며, 오토핫키를 통해서도 창 투명도를 제어할 수 있다. 모든 배경화면을 투명하게 만드는 방법도 가능하다.

## → 이거 모든 배경화면 투명하게 만들기 가능함

```c++
Ctrl + Mouse 3 누르면 투명화
Win + Mouse 3 누르면 원래대로
```
